# Music Recommendation System

## Overview
This project is a senior capstone project focused on building a music recommendation system using collaborative filtering techniques and the Spotify dataset.

## Features
- Recommends music based on the user's favorite artist
- Uses similarity-based collaborative filtering
- Processes and analyzes large datasets
- Displays a cluster for the recommended songs

## Technologies Used
- Python
- Spotify Dataset / Spotify API
- Pandas / NumPy / Scikit-learn / Matplotlib

## What I Learned
- Data preprocessing and feature engineering
- System integration and troubleshooting
- Working with APIs and large datasets
- Documentation and testing

## How to Run
1. Clone the repository
2. Install dependencies
3. Run the application/script in your oonsole by using the commands:
set FLASK_APP=app.py
set FLASK_ENV=development
flask run (for Windows)
export FLASK_APP=app.py
export FLASK_ENV=development
flask run (for Mac/Linux)
4. Run the http://127.0.0.1:5000
